## STAGE 50 — Per-platform post preview

Show a live preview of what the post will look like on each selected
platform before publishing. Character limits are already enforced;
this adds a visual mock.

### Scope

- Preview cards for X (tweet), Bluesky (post), Mastodon (toot),
  LinkedIn (share), Facebook (page post), Instagram (feed post)
- Rendered in the composer sidebar (or a tab) using the current
  version's content + first image + account avatar/handle
- Respects character truncation, link card placeholder, image aspect
  ratios per platform spec
